# Right-Click-Trick
[![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

Chrome-like context menu

## Usage
```
rightClickTrick(menus)
```

### menus[]
Type: `Array-Object`

- title
Type: `String`

- disabled
Type: `Boolean`
Disable the item

- onClick
Type: `Function`

- isDivider
Type: `Boolean`
Show only split lines
